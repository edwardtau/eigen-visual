Lesson 2: The Eigenvalue Recipe

For a 2×2 matrix, trace, determinant, and discriminant give a compact way to calculate eigenvalues and predict what kind of transformation you are seeing.

Back to explorer

Start with a 2×2 matrix

Write the matrix entries with letters first. This keeps the calculation pattern visible before numbers get involved.

Trace

tr(A) = a + d

The trace is the sum of the diagonal entries.

Determinant

det(A) = ad - bc

The determinant is the signed area scale factor.

Build the characteristic equation

Eigenvalues are the values of λ where A - λI collapses some nonzero direction. Algebraically, that means:

det(A - λI) = 0

For a 2×2 matrix, that determinant always simplifies to:

λ2 - tr(A)λ + det(A) = 0
  1. Find the trace: add the diagonal entries.
  2. Find the determinant: multiply diagonal entries, then subtract the off-diagonal product.
  3. Plug those two numbers into the quadratic equation above.

Use the discriminant

The discriminant tells you what kind of eigenvalues the quadratic has before you even solve it.

Δ = tr(A)2 - 4 det(A)

Δ > 0

Two distinct real eigenvalues.

Geometrically, there are two real eigen-directions.

Δ = 0

One repeated real eigenvalue.

The matrix may have one eigen-direction or a whole plane of them.

Δ < 0

Complex conjugate eigenvalues.

For a real plane visualization, this usually means rotation is involved.

Solve for eigenvalues

Once you have trace and determinant, use the quadratic formula:

λ = (tr(A) ± √Δ) / 2

Each eigenvalue is a scale factor. If λ = 3, vectors on that eigen-direction triple in length. If λ = -2, they double and flip through the origin.

Find eigenvectors

For each eigenvalue, plug it into A - λI and solve:

(A - λI)v = 0
  1. Choose one eigenvalue.
  2. Subtract it from the diagonal entries of A.
  3. Solve the resulting homogeneous system for a nonzero vector v.
  4. Any nonzero multiple of v is also an eigenvector.

Worked example

Use the same matrix from Lesson 1, then calculate its eigenvalues.

To find directions, solve (A - 3I)v = 0 and (A - I)v = 0. The first gives direction (1, 1); the second gives direction (1, -1).

Quick checklist

  1. Compute tr(A).
  2. Compute det(A).
  3. Compute Δ = tr(A)2 - 4det(A).
  4. Use λ = (tr(A) ± √Δ) / 2.
  5. For each eigenvalue, solve (A - λI)v = 0.
  6. Interpret eigenvalues as stretching/flipping factors along eigenvector directions.
