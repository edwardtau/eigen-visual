Lesson 3: Real Eigenvectors

A real eigenvector is a real direction that survives the transformation without turning. The matrix may stretch it, shrink it, or flip it, but the vector stays on the same line.

Back to explorer

What real eigenvectors mean

A nonzero real vector v is an eigenvector of A when applying the matrix only changes its length and possibly flips its direction.

Av = λv

v

The input direction.

It must be nonzero.

λ

The eigenvalue.

It tells how much the vector scales.

λv

The output vector.

It lands on the same line as v.

The geometric intuition

Most directions turn when a matrix transforms the plane. A real eigenvector is special because its whole line maps back onto itself. In the explorer, this is the direction where the transformed vector sits exactly on top of the dashed probe line.

How to calculate real eigenvectors

Lesson 2 found eigenvalues. Lesson 3 uses each eigenvalue to find its direction.

  1. Compute the real eigenvalues using trace, determinant, and the discriminant.
  2. Pick one eigenvalue λ.
  3. Form A - λI by subtracting λ from the diagonal entries.
  4. Solve (A - λI)v = 0 for a nonzero vector v.
  5. Repeat for each real eigenvalue.

If the discriminant is negative, the matrix has no real eigenvectors. That is the bridge to Lesson 4.

Calculation example

Use the same friendly matrix from the earlier lessons:

For λ = 3

(A - 3I)v = 0

A - 3I = [[-1, 1], [1, -1]], so -x + y = 0. Therefore y = x.

Eigenvector: v = (1, 1)

For λ = 1

(A - I)v = 0

A - I = [[1, 1], [1, 1]], so x + y = 0. Therefore y = -x.

Eigenvector: v = (1, -1)

Check the answer

Always verify by multiplying A by the candidate vector and checking whether the result equals λv.

What to look for in the explorer

  1. Drag the probe vector around the unit circle.
  2. Watch the transformed vector and the dashed probe line.
  3. If the transformed vector stays on the dashed line, you found an eigenline.
  4. Read the scale value along that line: this is the eigenvalue.
  5. If the transformed vector points through the origin in the opposite direction, the eigenvalue is negative.

Common mistakes

Using the zero vector

The zero vector always maps to zero, so it does not count as an eigenvector.

Forgetting multiples

If (1, 1) is an eigenvector, then (2, 2), (-5, -5), and every nonzero multiple also works.

Mixing eigenvalues

Each eigenvalue has its own equation (A - λI)v = 0. Solve them one at a time.
