Lesson 1: The Basics

Before looking for eigenvectors, start with the basic action of a linear transformation. It can be thought of as a function that takes an input vector and produces an output vector. In equation form, we write this as Ax = b.

A matrix transforms the plane

A matrix acts as a linear transformation of the plane. Starting from a reference plane with basis vectors e1 = (1, 0) and e2 = (0, 1), the matrix can stretch, shrink, rotate, or skew everything — depending on the entries inside matrix A. We can write the transformed e1 as T(e1) or A e1, and similarly T(e2) or A e2 for e2.

Watch where the basis vectors land

Because A is linear, knowing where the two basis vectors land tells us everything. Every other vector in the plane follows the same rule.

The columns of A tell the story

Matrix A is made from two column vectors: the first column is where e1 lands, and the second column is where e2 lands.

What Ax = b means

In equation form, we write this as Ax = b. Here, A is the transformation matrix, x is the original input vector, and b is the transformed output vector.
