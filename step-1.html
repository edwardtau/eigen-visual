Step 1: Ax = b

Before looking for eigenvectors, start with the basic action of a matrix: it transforms one vector into another.

Back to explorer

A matrix transforms the plane

In this visualization, a matrix is best understood as a linear transformation of the plane. We start with a two-dimensional reference plane with x- and y-axes. Its unit basis vectors are e1 = (1, 0) and e2 = (0, 1). Depending on the numbers in the matrix, the transformation can stretch, shrink, rotate, or skew the plane.

Watch where the basis vectors land

The transformation encoded by matrix A moves every point and vector in the reference plane to a new location on the transformed plane. By watching where the basis vectors land, we can see what the whole matrix does.

The columns of A tell the story

Matrix A is made from two column vectors: the first column is where e1 lands, and the second column is where e2 lands.

What Ax = b means

In equation form, we write this as Ax = b. Here, A is the transformation matrix, x is the original input vector, and b is the transformed output vector.

Example of A x = b

The inputs to the equation are matrix A and vector x. The output is vector b. So, A acts on x to give b. Let's see what that looks like with actual numbers.

In this example, the first entry of b is 2*3 + 1*1 = 7, and the second entry is 1*3 + 2*1 = 5. So the input vector x = (3, 1) becomes the output vector b = (7, 5).
